# Scaffolding / cognitive-load management

## What it is

Structuring tasks, explanations, and supports to control intrinsic, extraneous, and germane
cognitive load, so learners can process essential information without overloading working memory
[P19][P6]. Scaffolds include task simplification, step-wise guidance, worked examples,
segmentation, and — critically — *fading* of support as expertise grows [P20][P6].

## Evidence tier

**High.** Cognitive Load Theory synthesizes decades of experiments showing that designs which
reduce extraneous load and manage intrinsic load produce better learning and transfer [P19][P6].
Practitioner guides collate the classroom-ready effects (worked examples, goal-free problems,
modality effects) [P20]; the minimal-guidance critique gives the same conclusion from the failure
side [P21].

## When to use

- Introductory or high-complexity material (STEM, multi-step procedures) where element
  interactivity is high and learners lack schemas [P6][P7].
- Always paired with **fading**: high guidance → completion problems → open problem solving as
  expertise develops [P7][P19].
- Multimedia design choices (coherence, signaling, avoiding split attention) are load management
  too [P24][P25].

## Boundary conditions & pitfalls

- **Over-scaffolding**: support that outlives its need creates dependency, removes productive
  struggle, and triggers expertise reversal for advanced learners [P19][P7].
- **Load mismanagement**: redundant parallel explanations, or diagrams split from the text that
  explains them, *add* extraneous load [P26][P27].
- **The opposite failure**: minimal-guidance/pure-discovery instruction for novices overloads
  working memory and breeds misconceptions [P21] — see `socratic-guided-discovery.md` for the
  guided-questioning boundary.

## Worked example

A programming course introduces recursion by first showing a visual trace of a simple recursive
function, then a worked example with annotated stack frames. Learners next get partially
scaffolded problems where some steps are given and some must be filled in; finally scaffolds are
removed and learners write full recursive solutions with only minimal hints available.
