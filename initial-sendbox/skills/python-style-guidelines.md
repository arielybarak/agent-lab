# Skill: Python Style and Simplicity

**Trigger:** When generating, reviewing, or refactoring Python code.

**Directives:**
1. **Standard:** Follow the [Google Python Style Guide](https://google.github.io/styleguide/pyguide.html).
2. **Student-Centric Modifications:**
   - Clearly document types using type hints (`def func(a: int) -> str:`).
   - Use Google-style docstrings for functions, outlining Args and Returns.
   - Keep list comprehensions simple;
